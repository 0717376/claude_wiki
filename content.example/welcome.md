# Добро пожаловать 👋

Это стартовая страница вашей персональной вики.

- Создавайте страницы кнопкой **+** в дереве слева или прямо из чата: *«заведи заметку про …»*.
- Редактируйте по кнопке **Редактировать** — сохранение мгновенное, без кнопки «Сохранить».
- Спрашивайте ассистента справа — он ищет по вашим заметкам, отвечает и пишет страницы.

## Идеи, с чего начать

- `inbox/` — складывать сюда всё непонятное, разобрать потом.
- `projects/` — заметки по проектам.
- `personal/` — личное: рецепты, списки, мысли.

Удалите эту страницу, когда обживётесь.
